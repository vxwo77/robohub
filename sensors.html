Sensors Information

This page contains information about common sensors used in robotics.

HC-SR04

Distance sensor used to measure objects.

DHT22

Measures temperature and humidity.

PIR Sensor

Used to detect motion.

ADXL345

Accelerometer used to detect movement and tilt.

Actuators Table

Actuator Voltage Use
Servo Motor 5V Angle Control
DC Motor 5V-12V Movement
Relay Module 5V Switching Devices
Buzzer 5V Sound Alerts

Tips

  • Check voltage before connecting sensors.
  • Read sensor datasheets.
  • Use correct wiring.
  • Test sensors before using them in projects.
Always make sure the sensor voltage matches your board.
