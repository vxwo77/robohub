Hardware Information

This page contains information about common robotics boards.

Arduino Uno

Arduino Uno is one of the most popular boards for beginners. It is easy to use and supports many projects.

ESP32

ESP32 supports WiFi and Bluetooth communication. It is commonly used in IoT projects.

Raspberry Pi

Raspberry Pi is a small computer that can run Linux. It can be used for robotics and programming projects.
